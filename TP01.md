# 1. Netflix

**Qué problema resuelve con ML**
Netflix usa machine learning para **recomendar películas y series personalizadas** según lo que cada usuario mira, califica o abandona.

**Tipo de ML**

Aprendizaje supervisado.

**Por qué es supervisado**

El sistema se entrena con datos donde ya se conoce el comportamiento del usuario (qué contenido vio, si le gustó o no).
Con esa información aprende a **predecir qué contenido podría gustarle a un usuario en el futuro**.

**Datos que necesita**

* Historial de visualización
* Películas o series que el usuario terminó
* Calificaciones o likes
* Búsquedas realizadas
* Tiempo que se ve cada contenido

# 2. Spotify

**Qué problema resuelve con ML**
Spotify utiliza machine learning para **recomendar música y crear playlists personalizadas** según los gustos de cada usuario.

**Tipo de ML**

Aprendizaje no supervisado.

**Por qué es no supervisado**

El sistema analiza millones de canciones y comportamientos de usuarios para **encontrar patrones y similitudes entre canciones o gustos musicales**, sin que alguien indique manualmente qué canciones son parecidas.

**Datos que necesita**

* Canciones escuchadas
* Frecuencia de escucha
* Canciones que el usuario salta
* Playlists creadas
* Géneros musicales

---

# 3. Google Photos

**Qué problema resuelve con ML**
Permite **reconocer personas, animales, objetos y lugares en las fotos**, organizándolas automáticamente.

**Tipo de ML**

Aprendizaje supervisado.

**Por qué es supervisado**

Los modelos se entrenan con **millones de imágenes previamente etiquetadas** (por ejemplo: “perro”, “auto”, “persona”).
De esta manera el sistema aprende a **reconocer esos objetos en nuevas imágenes**.

**Datos que necesita**

* Imágenes
* Etiquetas de objetos o rostros
* Patrones visuales
* Metadatos de las fotos (fecha, ubicación)

# 4. Google Maps

**Qué problema resuelve con ML**
Google Maps utiliza machine learning para **predecir el tráfico y calcular la ruta más rápida** entre dos lugares.

**Tipo de ML**

Aprendizaje supervisado.

**Por qué es supervisado**

El sistema se entrena con **datos históricos de tráfico y tiempos de viaje reales**.
Con esos ejemplos aprende a **predecir cuánto tiempo tardará un recorrido según el tráfico actual**.

**Datos que necesita**

* Ubicación GPS de dispositivos
* Velocidad de los vehículos
* Historial de tráfico
* Hora del día
* Accidentes o cortes de calles

---

# 5. Mercado Libre (empresa latinoamericana)

**Qué problema resuelve con ML**
Mercado Libre usa machine learning para **recomendar productos, ordenar resultados de búsqueda y detectar fraudes en pagos**.

**Tipo de ML**

Aprendizaje supervisado.

**Por qué es supervisado**

Los modelos se entrenan con **datos de compras y transacciones pasadas**, donde ya se sabe si una compra fue normal o fraudulenta.
Con esa información el sistema aprende a **predecir comportamientos sospechosos o recomendar productos relevantes**.

**Datos que necesita**

* Historial de compras
* Búsquedas de productos
* Datos de pagos
* Comportamiento del usuario en la plataforma
* Ubicación o dispositivo
